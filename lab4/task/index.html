Завдання: 19. Розклад тренувань: Об'єкти: тренування з властивостями: вид тренування, тривалість, рівень складності, чи виконано.

1. Створено об’єкт тренування з властивостями: тип, тривалість, складність, виконано.

2. Додано методи для виведення інформації та позначення тренування як виконаного.

3. Реалізовано масив розкладу тренувань та функції для відображення, додавання, сортування й фільтрації.

4. Додано обчислення середньої тривалості тренувань.

5. Результати виводяться у консоль.
